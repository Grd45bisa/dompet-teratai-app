# Setup

This repository is meant to be opened with **Android Studio**.

1. Copy `local.properties.example` to `local.properties`
2. Set `sdk.dir=...`
3. Open the project in Android Studio and let it sync.

If Gradle wrapper is missing/broken in your environment, regenerate via Android Studio:
- File → New → Import Project / Sync Gradle
- Or run `gradle wrapper` (if you have gradle installed)
